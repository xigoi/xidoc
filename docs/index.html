xidoc

Introduction Manual Commands

xidoc is a consistent markup language that compiles to HTML and \mathrm\LaTeX. It's currently very work-in-progress, but I'm already using it for my school notes and this documentation.

A Quick Look

xidoc code

[section HTML;
  [p
    HTML is such an [it ugly] language!
    Each [ms <tag>] has to be repeated twice.
  ]
]
[section LaTeX;
  [p
    LaTeX is great for writing math formulas like [$ e^{\tau i} = 1],
    but it's painful to write with all those [ms \begin], [ms \end]
    and different syntaxes for different things.
  ]
]

Output

HTML

HTML is such an ugly language! Each <tag> has to be repeated twice.

LaTeX

LaTeX is great for writing math formulas like e^{\tau i} = 1, but it's painful to write with all those \begin, \end and different syntaxes for different things.
